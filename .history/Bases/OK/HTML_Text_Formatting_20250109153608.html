HTML_Colors

HTML colors are specified with predefined color names, or with RGB, HEX, HSL, RGBA, or HSLA values.

Colors Names

In HTML, a color can be specified by using a color name

Color Values

In HTML, colors can also be specified using RGB (value represents RED, GREEN, and BLUE light sources), HEX(hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color.), HSL (stands for hue, saturation, and lightness.), RGBA (value is an extension of RGB with an Alpha channel (opacity)). and HSLA (values are an extension of HSL with an Alpha channel (opacity)).

The following three elements have their background color set with RGB, HEX, and HSL values.

Ultramarine Blue
  • RGB (77,106,255)
  • HEX #4D6AFF
  • HSL (230, 100%, 65%)

The following two elements have their background color set with RGBA and HSLA values, which add an Alpha channel to the color (here we have 50% and 20% transparency)

  • RGBA (77,106,255, 0.5)
  • HSLA (230, 100%, 65%, 0.2)

Shades of Gray
